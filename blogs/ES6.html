    # 1. 箭头函数

    ES6中新增了箭头函数，它是一种更简洁的函数写法。箭头函数的语法比普通函数更加简洁，并且可以避免一些常见的问题，例如this指向的问题。

        const add = (a, b) => a + b;
    const add = (a, b) => {
        return a + b;       

    }

    # 7. 箭头函数的this指向

    箭头函数的this指向在函数定义时就已经确定，而不是在函数调用时确定。这意味着箭头函数的this指向不会随着函数的调用而改变。

        const person = {
        name: 'John',
        sayHello() {
            console.log(`Hello, my name is ${this.name}.`);
        }
    }

    # 9. 生成器

    ES6中新增了生成器，使得异步操作更加方便和直观。

        function* generateNumbers() {
        yield 1;
        yield 2;
        yield 3;    
    }

    const numbers = generateNumbers();
    console.log(numbers.next().value); // 输出 1
    console.log(numbers.next().value); // 输出 2   
    console.log(numbers.next().value); // 输出 3
